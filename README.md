# fcc-fastapi-demo

freeCodeCamp FastAPI [demo](https://youtu.be/0sOvCWFmrtA)

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install -r requirements.txt
```

## Usage

Run server locally

```bash
uvicorn main:app --reload
```

Enabled enpoints

```bash
# GET
../posts

# POST
../createpost

# Swagger UI
../docs
```
